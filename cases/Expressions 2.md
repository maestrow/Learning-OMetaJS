## Rule names in expression

```ometajs-eval
    rule1:r1 rule2 -> [r1] | rule3:r3 -> [r3]
```

## Strings

To specify string you have an options:

- `#string`
- ``string`

```ometajs-eval
    #string
    -> `string
```

## Symbol @

Похоже, что @ служил в качестве префикса для указания правила. Сейчас правило можно указывать без этого префикса.
